preprocessor directive - instruction that is processed by the preprocessor before the actual compilation of the program begins.


| Compiler                                               | Interpreter                                            |
| ------------------------------------------------------ | ------------------------------------------------------ |
| Translates the **entire program at once**              | Translates and executes **one line at a time**         |
| Generates an **executable file**                       | Does **not usually create a separate executable file** |
| Errors are shown **after compiling the whole program** | Errors are shown **line by line during execution**     |
| Execution is generally **faster**                      | Execution is generally **slower**                      |
| Example: **GCC**                                       | Example: **Python Interpreter**                        |


C++ is different from C in several ways:

C is a procedural language, while C++ is an object-oriented language (it also supports procedural programming).
C++ supports classes and objects, while C does not.
C++ provides inheritance, polymorphism, and encapsulation, while C does not support these object-oriented features.
C++ supports function overloading, while C does not.
C++ has features like constructors and destructors, which are not available in C.
C++ generally offers better support for code reusability and larger projects.


If you do not use using namespace std;, you can still use cout and cin by writing std:: before them because cout and cin belong to the std (standard) namespace.
#include <iostream>

int main() {
    std::cout << "Enter a number: ";

    int x;
    std::cin >> x;

    std::cout << "You entered: " << x;

    return 0;
}


Data Types in C++
Built-In (Primitive)- int, char, float, double, void
Derived - arrays, pointers, refrences
User-Defined - structures, unions, classes, enumerations.

| ---------------------------- | --------------------------------: | --------- |

| Double quotes `" "`          | A string (sequence of characters) | `"Anmol"` |
